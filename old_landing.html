Enterprise-Grade Security

Your credentials and trading data are protected with military-grade encryption and best-in-class secure infrastructure practices.

🔐

AES-256 Fernet Encryption

All API credentials — including Fyers Client ID, Secret, and Access Tokens — are encrypted at rest in the database using AES-256 Fernet symmetric encryption. Keys are dynamically generated and stored in isolated environment variables on server boot.

🛡️

HTTPS / TLS 1.3 in Transit

All traffic between your browser and our servers is encrypted end-to-end via Let's Encrypt SSL/TLS 1.3 certificates. No data is ever transmitted in plain text. HTTP traffic is automatically redirected to HTTPS on port 443.

🔑

Session Cookie Security

Authentication sessions use scoped HttpOnly cookies bound to the root path (/), preventing cross-site script access. Sessions are server-side invalidated on logout, and idle sessions are automatically expired after 15 minutes of inactivity.

⏱️

Idle Auto-Lock (15 Minutes)

The platform detects inactivity by monitoring mouse, keyboard, scroll, and touch events. After 15 continuous minutes of no activity, the session is automatically locked and the user is redirected to the login page for re-authentication.

👥

Role-Based Access Control

Multi-user platform with segregated access. Standard users can only access their own data, positions, and API credentials. Administrators have additional privileges to manage user accounts, passwords, and platform settings from the Admin Panel.

🖥️

Isolated GCP Infrastructure

Hosted on Google Cloud Platform (GCP) in the Asia South 1 (Mumbai) region. The server runs under a dedicated systemd process with strict file permissions. All environment secrets are managed via isolated .env files, never version-controlled.

✅

SSL Certificate Verified — controlntrading.online

Your connection is fully secured with a Let's Encrypt SSL certificate covering both controlntrading.online and www.controlntrading.online. Certificate is auto-renewed every 90 days via Certbot.

Regulatory Compliance

ControlN Trading is built with responsible trading practices, risk controls, and full compliance with SEBI's algorithmic trading guidelines for retail investors.

📋

SEBI Algo Trading Guidelines

All algorithmic strategies comply with SEBI's framework for retail algorithmic trading. Trades are executed via SEBI-registered broker APIs (Fyers Securities) with full audit trails and order logging.

🧾

Risk Parameter Controls

Platform enforces mandatory risk limits including Max Daily Loss (MDL), Max Daily Trades, and Position Profit Targets. All limits are configurable per user and enforced automatically before any order is placed.

📊

Full Trade Audit Trail

Every trade entry, modification, exit, and square-off is logged with timestamps and order IDs. Full P&L history with broker-simulated charges (STT, brokerage, GST) is exportable as CSV reports.

🏦

Broker Integration via API

All orders are routed exclusively through Fyers Securities, a SEBI-registered stock broker. Credentials are managed by the user directly and never shared with third parties. No pooling of funds.

🔒

Data Privacy & Retention

No personal financial data is shared with third parties. Trade and user data is stored in isolated SQLite databases per deployment, with encrypted credentials. Users may request account deletion at any time.

⚠️

Emergency Exit Controls

One-click Emergency Exit button immediately squares off all open positions and disables automation. Available 24/7 from the dashboard header regardless of active strategy state.
